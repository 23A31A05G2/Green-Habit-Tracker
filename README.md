# Green Habit Tracker 🌱

## Overview

Green Habit Tracker is a web application designed to encourage sustainable living by helping users track eco-friendly habits and monitor their positive environmental impact.

## Features

* User Login
* Habit Tracking
* Carbon Footprint Reduction Tracking
* Progress Bar
* Achievement Badges
* Custom Habit Creation
* Local Storage Support

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Objective

To motivate users to adopt environmentally friendly habits and contribute to climate action through daily sustainable practices.
